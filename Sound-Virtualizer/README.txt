# Sound Virtualizer
**Created by nghhai1090**

## Dependencies:
First you have to make sure that you have these packages installed. You can install them using this command:

```pip install <package_name>```

`pip` is a package manager for python.

* tkinter
* time
* pyaudio
* numpy
* wave

**Notes:**
* `time` should be installed by default
* If you ran at an issue while trying to install pyaudio you can install it using the folloing command: `sudo apt install python3-pyaudio` however this only works for debian based distros if you are on another OS, try searching online for a solution.
